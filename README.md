# 💼 Invoice Generator

A professional, feature-rich desktop application built with Electron for creating, customizing, and exporting invoices. Generate beautiful invoices with ease using an intuitive interface with real-time preview.

## ✨ Features

### Core Functionality
- ✅ **Professional Invoice Creation** - Create professional invoices with customizable templates
- ✅ **Real-Time Preview** - See your invoice update instantly as you type
- ✅ **Multiple Export Formats** - Export to PDF, HTML, or print directly
- ✅ **Dark Mode Support** - Eye-friendly dark theme with automatic persistence
- ✅ **Responsive Design** - Works seamlessly on different screen sizes

### Company Customization
- 🏢 Company logo upload (PNG, JPG, GIF, WebP)
- 📝 Company name, address, contact information
- 📧 Email and phone number
- 🌐 Website URL
- 💰 Tax ID / GST / VAT Number

### Client Management
- 👤 Client name and company
- 📍 Client address and contact information
- 📧 Email and phone number
- 📱 Multiple contact options

### Invoice Details
- 📄 Custom invoice numbers
- 📅 Invoice and due dates
- 💱 Multiple currency support (USD, EUR, GBP, INR, CAD, AUD)
- 🧮 Automatic tax calculation
- 🏷️ Discount percentage support
- 📋 Unlimited invoice items

### Invoice Items
- 📦 Item description
- 🔢 Quantity with decimal support
- 💵 Unit price configuration
- 🧮 Per-item tax percentage
- 📊 Automatic total calculation

### Additional Features
- 📌 Custom notes section
- ⚖️ Terms and conditions support
- ✍️ Digital signature upload
- 🖼️ Professional invoice formatting
- 💾 Save as HTML file
- 🖨️ Direct print functionality
- 📥 PDF generation with html2pdf

### UI/UX Features
- 🎨 Modern glassmorphism design
- 🌓 Dark/Light theme toggle
- ⌨️ Keyboard-friendly interface
- 🎯 Intuitive form layout
- 📱 Mobile-responsive design
- ✨ Smooth animations and transitions

## 🛠️ Technology Stack

- **Electron** - Desktop application framework
- **HTML5** - Structure and markup
- **CSS3** - Modern styling with glassmorphism
- **Vanilla JavaScript** - Core application logic
- **html2canvas** - Canvas rendering for PDF
- **jsPDF** - PDF generation library
- **Node.js** - Backend runtime
